# Visual Dark Fantasy

> Skill de identidade visual: todos os outputs devem seguir a estética dark fantasy medieval do Grimoire Studio.

## Paleta de Cores

Use SEMPRE estas cores como referência nos outputs visuais e descrições:

- **Pedra escura** `#1a1510` — fundo principal
- **Dourado envelhecido** `#c8942a` / `#e8b84a` — destaques, títulos
- **Vermelho sangue** `#8b1a1a` — ações, urgência, CTAs
- **Osso** `#c8bca0` — texto principal
- **Runa azul** `#60a8c8` — informações, links
- **Veneno verde** `#50a040` — sucesso, confirmações

## Tipografia

- Títulos: **Cinzel** — serifa clássica com peso medieval
- Corpo: **Press Start 2P** — pixel art retro
- Hierarquia: títulos em dourado, subtítulos em osso, descrições em dim `#806858`

## Tom de Linguagem

- Use termos medievais ocasionais: *forjar*, *conjurar*, *invocar*, *grimório*, *runa*, *saga*, *brasão*
- Evite linguagem corporativa genérica
- Seja direto e épico ao mesmo tempo
- Projetos = "sagas", tarefas = "missões", clientes = "patronos"

## Regras Visuais

Quando sugerir layouts, componentes ou elementos visuais:
- Preferir bordas finas com detalhes em dourado
- Usar espaçamento generoso com áreas de sombra intensa
- Texturas de pedra ou pergaminho como fundos
- Ícones geométricos simples (losangos, cruzes, triângulos) no lugar de ícones modernos
- Animações lentas e pesadas, não rápidas e "snappy"
